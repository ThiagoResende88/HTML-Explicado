HTML Explicado

HTML Basic 1

O HTML (HyperText Markup Language) é a linguagem padrão para criar páginas da web. Ele define a estrutura básica de uma página web, usando tags para indicar diferentes tipos de conteúdo, como cabeçalhos, parágrafos, listas, etc.

<!DOCTYPE html>
<html>
<head>
    <title>Título da Página</title>
</head>
<body>
    <h1>Título Principal</h1>
    <p>Parágrafo de texto.</p>
</body>
</html>

HTML Elements

Os elementos HTML são os blocos de construção básicos de uma página da web. Eles são representados por tags e podem incluir texto, imagens, links, formulários e muito mais.

<!DOCTYPE html>
<html>
<head>
    <title>Título da Página</title>
</head>
<body>
    <h1>Título Principal</h1>
    <p>Parágrafo de texto.</p>
</body>
</html>

HTML Attributes

Os atributos HTML fornecem informações adicionais sobre elementos HTML. Eles são sempre especificados na tag de abertura e podem conter valores que modificam o comportamento ou a aparência do elemento.

<!DOCTYPE html>
<html>
<head>
    <title>Título da Página</title>
</head>
<body>
    <h1>Título Principal</h1>
    <p>Parágrafo de texto.</p>
</body>
</html>

HTML Headings

As headings (cabeçalhos) são usadas para definir a estrutura e o conteúdo de uma página da web. Existem seis níveis de cabeçalhos, que vão de <h1> (o mais importante) a <h6> (o menos importante).

<!DOCTYPE html>
<html>
<head>
    <title>Título da Página</title>
</head>
<body>
    <h1>Título Principal</h1>
    <p>Parágrafo de texto.</p>
</body>
</html>

HTML Paragraphs

Os parágrafos são usados para organizar e estruturar o conteúdo textual de uma página da web. Eles são representados pela tag <p> e são separados uns dos outros por uma quebra de linha.

<!DOCTYPE html>
<html>
<head>
    <title>Título da Página</title>
</head>
<body>
    <h1>Título Principal</h1>
    <p>Parágrafo de texto.</p>
</body>
</html>

6. HTML Styles

Os estilos em HTML são usados para controlar a aparência e o layout dos elementos em uma página da web. Eles podem ser aplicados inline, internamente ou externamente, usando a tag <style> ou arquivos CSS externos.

<!DOCTYPE html>
        <html>
        <head>
            <style>
                h1 {
                    color: red;
                    font-size: 24px;
                }
            </style>
        </head>
        <body>
            <h1>Estilizado em vermelho</h1>
        </body>
        </html>

7. HTML Formatting

A formatação em HTML permite controlar a aparência do texto e outros elementos em uma página da web. Isso inclui negrito (<strong>), itálico (<em>), sublinhado (<u>), entre outros.

<!DOCTYPE html>
        <html>
        <body>
            <p>Este texto está em <strong>negrito</strong> e <em>itálico</em>.</p>
        </body>
        </html>

8. HTML Quotations

As citações em HTML são usadas para marcar citações de outras fontes. Existem dois tipos de citações: inline, usando <q>, e blockquote, para citações longas, usando <blockquote>.

<!DOCTYPE html>
        <html>
        <body>
            <p>Ela disse <q>Olá!</q> e continuou seu caminho.</p>
            <blockquote>Este é um exemplo de citação longa.</blockquote>
        </body>
        </html>

9. HTML Comments

Os comentários em HTML são usados para inserir notas ou comentários no código HTML que não são exibidos na página da web. Eles são úteis para documentar o código ou desativar temporariamente partes do código.

<!DOCTYPE html>
        <html>
        <body>
            <!-- Este é um comentário em HTML -->
            <p>Este texto será exibido na página.</p>
        </body>
        </html>

10. HTML Colors

As cores em HTML podem ser especificadas de várias maneiras, incluindo nomes de cores, códigos hexadecimais, RGB, RGBA e HSL. Elas são usadas para estilizar elementos, como texto, fundos, bordas, etc.

<!DOCTYPE html>
        <html>
        <body style="background-color: #ff0000;">
            <h1 style="color: #ffffff;">Texto em branco sobre fundo vermelho</h1>
        </body>
        </html>

11. HTML Links

Os links em HTML são usados para criar links para outras páginas da web, arquivos, endereços de e-mail, entre outros. Eles são definidos pela tag <a> e têm um atributo href que especifica o destino do link.

<!DOCTYPE html>
        <html>
        <body>
            <a href="https://www.example.com">Link para example.com</a>
        </body>
        </html>

12. HTML Favicon

O favicon em HTML é o ícone pequeno exibido ao lado do título da página na barra de endereços do navegador. Ele é especificado na tag <link> dentro do elemento <head>.

<!DOCTYPE html>
        <html>
        <head>
            <link rel="icon" href="favicon.ico" type="image/x-icon">
        </head>
        <body>
            <h1>Página com favicon</h1>
        </body>
        </html>

13. HTML Table

As tabelas em HTML são usadas para exibir dados em linhas e colunas. Elas são criadas usando as tags <table> para a tabela em si, <tr> para as linhas e <td> para as células.

<!DOCTYPE html>
        <html>
        <body>
            <table border="1">
                <tr>
                    <td>Célula 1</td>
                    <td>Célula 2</td>
                </tr>
                <tr>
                    <td>Célula 3</td>
                    <td>Célula 4</td>
                </tr>
            </table>
        </body>
        </html>

14. HTML List

As listas em HTML são usadas para organizar e estruturar informações em forma de lista. Existem três tipos principais de listas: ordenadas (<ol>), não ordenadas (<ul>) e de definição (<dl>).

<!DOCTYPE html>
        <html>
        <body>
            <ul>
                <li>Item 1</li>
                <li>Item 2</li>
            </ul>
        </body>
        </html>

15. HTML Block & Inline

Os elementos em HTML são classificados como elementos de bloco ou elementos inline. Elementos de bloco começam em uma nova linha e ocupam todo o espaço horizontal disponível, enquanto elementos inline são exibidos em linha, ocupando apenas o espaço necessário.

<!DOCTYPE html>
        <html>
        <body>
            <div>Elemento de bloco</div>
            <span>Elemento inline</span>
        </body>
        </html>

16. HTML Div

A tag <div> em HTML é usada como um contêiner genérico para agrupar elementos e aplicar estilos a eles usando CSS. Ela não possui significado semântico próprio e é usada principalmente para organização e estruturação do layout da página.

<!DOCTYPE html>
        <html>
        <body>
            <div style="background-color: lightblue; padding: 10px;">
                <p>Conteúdo dentro da div</p>
            </div>
        </body>
        </html>

17. HTML Classes

As classes em HTML são usadas para aplicar estilos a um grupo de elementos com características semelhantes. Elas são definidas usando o atributo class e podem ser estilizadas com CSS.

<!DOCTYPE html>
        <html>
        <head>
            <style>
                .destaque {
                    font-weight: bold;
                    color: red;
                }
            </style>
        </head>
        <body>
            <p class="destaque">Texto em destaque</p>
        </body>
        </html>

18. HTML Id

O atributo id em HTML é usado para identificar exclusivamente um elemento na página. Ele é semelhante à classe, mas é usado para um único elemento específico, enquanto a classe pode ser usada em vários elementos.

<!DOCTYPE html>
        <html>
        <body>
            <h1 id="titulo">Título</h1>
            <p>Parágrafo com <a href="#titulo">link para o título</a>.</p>
        </body>
        </html>

19. HTML Iframes

Os iframes em HTML são usados para incorporar páginas web ou conteúdo de outras fontes dentro de uma página. Eles são representados pela tag <iframe> e podem exibir conteúdo de diferentes domínios.

<!DOCTYPE html>
    <html>
    <body>
        <iframe src="https://www.example.com"></iframe>
    </body>
    </html>

20. HTML JavaScript

O JavaScript em HTML é usado para adicionar interatividade e dinamismo a uma página da web. Ele pode ser incluído no código HTML usando a tag <script> e permite manipular elementos, responder a eventos e muito mais.

<!DOCTYPE html>
    <html>
    <body>
        <button onclick="alert('Olá, mundo!')">Clique aqui</button>
    </body>
    </html>

21. HTML Semantics

A semântica em HTML se refere ao significado dos elementos e tags em uma página da web. Ela ajuda os navegadores e motores de busca a entenderem melhor o conteúdo e a estrutura da página, o que pode melhorar a acessibilidade e a indexação.

<!DOCTYPE html>
        <html>
        <body>
            <header>
                <h1>Título do Site</h1>
                <nav>
                    <a href="pagina1.html">Página 1</a>
                    <a href="pagina2.html">Página 2</a>
                </nav>
            </header>
            <main>
                <article>
                    <h2>Título do Artigo</h2>
                    <p>Conteúdo do artigo.</p>
                </article>
            </main>
            <footer>
                <p>Rodapé do site.</p>
            </footer>
        </body>
        </html>

22. HTML Style Guide

O guia de estilo em HTML é um conjunto de diretrizes e recomendações para escrever código HTML de forma consistente e organizada. Ele pode incluir convenções de nomenclatura, formatação de código, uso de espaços em branco, entre outros.

<!DOCTYPE html>
        <html>
        <body>
            <h1>Título Principal</h1>
            <p>Parágrafo de texto.</p>
        </body>
        </html>

23. HTML Entities

As entidades em HTML são códigos especiais usados para representar caracteres especiais que não são facilmente digitados no teclado. Elas são precedidas pelo caractere "&" e terminam com ";".

<!DOCTYPE html>
        <html>
        <body>
            <p>Este é um exemplo de entidade: &copy;</p>
        </body>
        </html>

24. HTML Symbols

Os símbolos em HTML são usados para exibir caracteres especiais, como setas, símbolos matemáticos, emojis, entre outros. Eles são representados por códigos específicos e são úteis para adicionar elementos visuais às páginas da web.

<!DOCTYPE html>
        <html>
        <body>
            <p>Este é um exemplo de símbolo: &#9658;</p>
        </body>
        </html>

25. HTML Emojis

Os emojis em HTML são usados para adicionar emoticons e símbolos gráficos às páginas da web. Eles são representados por códigos específicos e podem ser usados para transmitir emoções e sentimentos.

<!DOCTYPE html>
        <html>
        <body>
            <p>Este é um exemplo de emoji: &#128512;</p>
        </body>
        </html>

26. HTML Charsets

O conjunto de caracteres em HTML especifica a codificação de caracteres usada para exibir o texto de uma página da web. Ele é definido pelo elemento <meta charset="utf-8"> dentro da tag <head>.

<!DOCTYPE html>
    <html>
    <head>
        <meta charset="utf-8">
    </head>
    <body>
        <p>Texto com caracteres especiais.</p>
    </body>
    </html>

27. HTML URL Encode

A codificação de URL em HTML é usada para converter caracteres especiais em uma URL em uma sequência segura para transmissão pela Internet. Ela é feita substituindo os caracteres especiais por códigos específicos.

<!DOCTYPE html>
    <html>
    <body>
        <a href="https://www.example.com/search?q=espaço%20em%20branco">Link com URL codificada</a>
    </body>
    </html>

28. HTML vs XHTML

O HTML e o XHTML são linguagens de marcação para a criação de páginas da web. O HTML é mais flexível e permite certas práticas mais relaxadas, enquanto o XHTML é uma versão mais rigorosa e estruturada do HTML.

<!DOCTYPE html PUBLIC "-//W3C//DTD XHTML 1.0 Transitional//EN"
        "http://www.w3.org/TR/xhtml1/DTD/xhtml1-transitional.dtd">
        <html xmlns="http://www.w3.org/1999/xhtml">
        <head>
            <title>Título da Página</title>
        </head>
        <body>
            <h1>Título Principal</h1>
            <p>Parágrafo de texto.</p>
        </body>
        </html>

29. HTML Forms

Os formulários em HTML são usados para coletar dados dos usuários, como informações de contato, login, etc. Eles são criados usando a tag <form> e podem conter vários tipos de campos, como texto, caixas de seleção, botões, entre outros.

<!DOCTYPE html>
        <html>
        <body>
            <form action="/submit" method="post">
                <label for="name">Nome:</label>
                <input type="text" id="name" name="name"><br>
                <input type="submit" value="Enviar">
            </form>
        </body>
        </html>

30. HTML Form Attributes

Os atributos de formulário em HTML são usados para definir diferentes características e comportamentos dos formulários. Alguns dos atributos mais comuns incluem action, method, name, id, enctype, entre outros.

<!DOCTYPE html>
        <html>
        <body>
            <form action="/submit" method="post" name="myForm" id="myForm" enctype="multipart/form-data">
                <!-- Campos do formulário aqui -->
            </form>
        </body>
        </html>

31. HTML Form Elements

Os elementos de formulário em HTML são usados para criar campos interativos em um formulário. Alguns dos elementos mais comuns incluem <input>, <textarea>, <select>, <button>, entre outros.

<!DOCTYPE html>
        <html>
        <body>
            <form action="/submit" method="post">
                <label for="username">Username:</label>
                <input type="text" id="username" name="username"><br>
                <label for="password">Password:</label>
                <input type="password" id="password" name="password"><br>
                <input type="submit" value="Submit">
            </form>
        </body>
        </html>

32. HTML Input Types

Os tipos de entrada em HTML são usados para especificar o tipo de dados que um campo de formulário pode aceitar. Alguns dos tipos de entrada mais comuns incluem text, password, checkbox, radio, submit, entre outros.

<!DOCTYPE html>
        <html>
        <body>
            <form action="/submit" method="post">
                <label for="email">Email:</label>
                <input type="email" id="email" name="email"><br>
                <label for="birthdate">Birthdate:</label>
                <input type="date" id="birthdate" name="birthdate"><br>
                <input type="submit" value="Submit">
            </form>
        </body>
        </html>

33. HTML Input Attributes

Os atributos de entrada em HTML são usados para adicionar diferentes características e comportamentos aos campos de formulário. Alguns dos atributos mais comuns incluem placeholder, required, disabled, value, entre outros.

<!DOCTYPE html>
        <html>
        <body>
            <form action="/submit" method="post">
                <label for="name">Name:</label>
                <input type="text" id="name" name="name" placeholder="Enter your name" required><br>
                <input type="submit" value="Submit">
            </form>
        </body>
        </html>

34. Input Form Attributes

Os atributos de formulário de entrada em HTML são usados para adicionar diferentes características e comportamentos aos campos de formulário. Alguns dos atributos mais comuns incluem autocomplete, autofocus, form, formaction, entre outros.

<!DOCTYPE html>
        <html>
        <body>
            <form action="/submit" method="post">
                <label for="username">Username:</label>
                <input type="text" id="username" name="username" autocomplete="username" autofocus><br>
                <input type="submit" value="Submit">
            </form>
        </body>
        </html>

35. HTML Canvas

O elemento <canvas> em HTML é usado para desenhar gráficos, animações, gráficos interativos e outros gráficos visuais em uma página da web. Ele fornece uma área de desenho dinâmica que pode ser manipulada com JavaScript.

<!DOCTYPE html>
        <html>
        <body>
            <canvas id="myCanvas" width="200" height="100" style="border:1px solid #000;"></canvas>
            <script>
                var canvas = document.getElementById('myCanvas');
                var ctx = canvas.getContext('2d');
                ctx.fillStyle = 'red';
                ctx.fillRect(10, 10, 50, 50);
            </script>
        </body>
        </html>
